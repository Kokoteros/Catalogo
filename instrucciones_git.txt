1) Clonamos el proyecto ulizando el comando git clone:
git clone https://github.com/Kokoteros/Catalogo.git

2) Podemos ver el estado de un proyecto usando
git status

3) Para agregar cambios
git add nombre_archivo o archivos
o bien
 git add . (agregamos todos los archivos)

 4) Comiteamos los cambios 
 git commit -m "un mensaje"

 5) Subimos los cambios a github con:
 git push
